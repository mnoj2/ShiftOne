# ShiftOne

ShiftOne is a full-stack Shift Management application built to track and manage worker attendance, shifts, and team analytics across three role-based dashboards — Admin, Supervisor, and Worker.

## Tech Stack
- **Frontend**: Angular 20
- **Backend**: .NET 8
- **Database**: SQL Server

## Key Features
- **Admin**: Complete user control and OCR-based form scanning functionalities.
- **Supervisor**: Access to team analytics, daily attendance summaries, and date-range history tracking.
- **Worker**: Simple tracking for signing in, signing off, manual adjustments, and viewing personal history.
